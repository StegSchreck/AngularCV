Sobre esta página AngularCV: Un CV simple y autohospedado

Esta página web ha sido creada como un proyecto de GitHub con el propósito de tener un CV en línea que pudiera ser alojado por cualquiera en su servidor. Aun así, el público principal a quien está dirigido es a los desarrolladores de software. Para construirlo, se han utilizado básicamente las siguientes tecnologías:

Angular Material Design GitHub CSS TypeScript JetBrains WebStorm AWS

Además, también se han incluído las siguientes fuentes de terceros:

IcoMoon icon font Roboto font Roboto Slab font jsPDF VER PROYECTO EN GITHUB
